α = 0.4187: IPCA esperado quando desemprego = 0 (intercepto teórico).

β = 0.0037: para cada +1 p.p. de desemprego, o IPCA mensal varia em média 0.0037 p.p. → relação positiva (estagflação, padrão Brasil 2015-2017).
